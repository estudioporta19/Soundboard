Como Usar:

  • Arrastar e Largar: Arraste ficheiros de áudio (MP3, WAV, OGG) para as células para as preencher.
  • Clicar: Clique numa célula vazia para abrir um diálogo de seleção de ficheiro. Clique numa célula preenchida para reproduzir o som.
  • Atalhos de Teclado: Pressione a tecla correspondente no seu teclado para reproduzir o som. (Ex: Q para a primeira célula).
  • Navegação (Modo QLab): Pressione Espaço para tocar o próximo som disponível. Pressione Ctrl + Espaço para tocar o som disponível anterior. Células vazias são ignoradas.
  • Parar Sons: Pressione ESC para parar todos os sons a tocar.
  • Ajustar Volume: Use o slider de volume ou as teclas ⬆️ e ⬇️ para controlar o volume global.
  • Apagar Som: Clique no ❌ no canto superior direito de uma célula para a esvaziar. *Um clique rápido apaga; um clique longo (>0.5s) faz fade out.*
  • Substituir Som: Clique no upload_file para carregar um novo som para a célula.
  • Mudar Nome: Clique no nome do som para editá-lo.
  • Controlar Fade In: Use o slider de Fade In, ou as teclas Ctrl + teclas numéricas 0-9 para definir a duração do fade in em segundos.
  • Controlar Fade Out: Use o slider de Fade Out, ou as teclas numéricas 0-9 para definir a duração do fade out em segundos.
  • Modo Reproduzir Múltiplos: Permite que vários sons toquem ao mesmo tempo se a caixa estiver marcada.
  • Modo Auto-Kill Anterior: Ao tocar um novo som, o som anteriormente ativo (se houver) será parado com um fade out rápido.
  • CUE / GO: Pressione Ctrl + Enter para 'cue' (marcar) um som. Pressione Enter para tocar todos os sons em 'cue' com fade-in. Pressione Shift + Enter para parar todos os sons em 'cue' com fade-out.
  • CUE Individual: Pressione Ctrl + clique na célula para adicionar/remover um som do 'cue'.
  • Remover CUE: Pressione Alt + Enter para remover todos os sons do 'cue' sem os parar.
